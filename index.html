シェアスケジュールへようこそ

データを Google スプレッドシートで管理します。
まず Google Apps Script のウェブアプリ URL を設定してください。

📋 セットアップ手順
  1. Google スプレッドシートを新規作成（名前: Schedule DB）
  2. gas/Code.gs の内容を Apps Script に貼り付け
  3. スプレッドシートの ID を SPREADSHEET_ID に設定
  4. ウェブアプリとしてデプロイ（アクセス: 全員）
  5. 表示された URL を下に入力
⚙️ 設定
URLを変更すると接続確認を行います
データの同期について
変更は自動で Google スプレッドシートに保存されます。
二人が同時に操作した場合は後に保存した方が優先されます。
